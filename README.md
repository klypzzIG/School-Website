# D.P. Public School AI-Style Interview Portfolio

A modern, animated personal portfolio website made with HTML, CSS, and JavaScript for a D.P. Public School interview. The school content uses stable information from the official DPPS website and avoids temporary details like toppers, notices, and one-time functions.

## Source files

- `dpps-portfolio.html` — the all-in-one file you can copy anywhere and use by itself.
- `index.html` — the multi-file page structure, sections, editor dialog, and placeholders that JavaScript fills.
- `styles.css` — the dark futuristic theme, layout, responsiveness, animations, editor styling, and cards.
- `content.js` — the main file you edit for your name, projects, school-fit answers, skills, and contact details.
- `script.js` — renders `content.js`, powers the Customize editor, mobile menu, AI terminal, cursor glow, and scroll reveal effects.

## How to update it yourself

### Easy method: use the Customize button

1. Open `dpps-portfolio.html` in a browser for the one-file version, or `index.html` for the multi-file version.
2. Click **Customize** in the navigation bar.
3. Edit the JSON text carefully.
4. Click **Save Updates**.
5. Your changes are saved in that browser using local storage.
6. Click **Reset** if you want to return to the original DPPS template.

### Permanent method: edit the content directly

1. For the one-file version, open `dpps-portfolio.html` in Notepad, VS Code, or any text editor and search for `const portfolioContent`.
2. For the multi-file version, open `content.js` in Notepad, VS Code, or any text editor.
3. Change `student.name`, `student.email`, `student.phone`, `student.city`, and `student.intro`.
4. Update `schoolFit`, `skills`, and `projects` with your real answers.
5. Save the file and reopen `dpps-portfolio.html` or `index.html`.

## Step-by-step: run it on a laptop

### Method 1: Open directly, fastest for interview practice

1. Easiest option: copy only `dpps-portfolio.html` to your laptop. Multi-file option: copy the full `School-Website` folder to your laptop.
2. For the one-file option, double-click `dpps-portfolio.html`. For the multi-file option, open the folder and make sure `index.html`, `styles.css`, `script.js`, `content.js`, and `README.md` are together.
3. Double-click `dpps-portfolio.html` or `index.html`, depending on which option you are using.
4. Choose Chrome, Edge, Firefox, or Safari if your laptop asks which app to use.
5. Click **Customize** in the website navigation if you want to update your name, contact details, skills, or projects temporarily in that browser.
6. For permanent edits, open `content.js` in Notepad or VS Code, change your details, save the file, and refresh the browser.

### Method 2: Run with a local server, best for phone testing

1. Open a terminal in the `School-Website` folder.
2. Run `python3 -m http.server 4173`. On some Windows laptops, use `python -m http.server 4173` instead.
3. Open `http://127.0.0.1:4173` on the laptop browser.
4. Keep the terminal window open while using the website.
5. Press `Ctrl + C` in the terminal when you want to stop the server.

## Step-by-step: run it on a phone

### Method 1: Copy the folder to your phone

1. Connect your phone to your laptop with a USB cable.
2. On the laptop, choose file transfer mode if your phone asks for permission.
3. Copy `dpps-portfolio.html` to your phone storage, or copy the full `School-Website` folder if you want the multi-file version.
4. On your phone, open the file manager app.
5. Tap `dpps-portfolio.html`. If you copied the folder version, open the `School-Website` folder and tap `index.html`.
6. Choose Chrome, Edge, Safari, or another browser.
7. If the page opens but styling is missing, check that `styles.css`, `script.js`, and `content.js` are in the same folder as `index.html`.

### Method 2: Open from your laptop over Wi-Fi

1. Connect your phone and laptop to the same Wi-Fi network.
2. On the laptop, open a terminal in the `School-Website` folder.
3. Run `python3 -m http.server 4173` or `python -m http.server 4173`.
4. Find your laptop IP address. Common examples look like `192.168.1.5` or `192.168.0.12`.
5. On your phone browser, open `http://YOUR-LAPTOP-IP:4173`, replacing `YOUR-LAPTOP-IP` with the real IP address.
6. Example: if your laptop IP is `192.168.1.5`, open `http://192.168.1.5:4173`.
7. Keep the laptop terminal open while using the website on your phone.

## Step-by-step: copy it to a pen drive

1. Insert the pen drive into your laptop.
